Cloud Sync with
Supabase
FileVault uses Supabase — a free, open-source cloud platform — to back up your files and sync across multiple devices. Supabase has a generous free tier that's more than enough for a school.
1
Create a Free Account
Visit supabase.com and sign up for free
2
Create a New Project
Takes about 2 minutes — pick any region
3
Run the SQL Setup
Copy the SQL from Cloud Sync → SQL Setup and run it once
4
Enter Your Credentials
Paste your Project URL and Anon Key in Cloud Sync settings
